## Requirements

- **Node.js** `>= 18.12.1`
- **Nuxt** `>= 3.13.0`

## Installation

Using [pnpm](https://pnpm.io):

```bash
pnpm add @kikiutils/nuxt @kikiutils/shared @kikiutils/types @nuxtjs/robots @unocss/nuxt @vueuse/nuxt unplugin-fonts
```

You can also use `yarn`, `npm`, or `bun`.
